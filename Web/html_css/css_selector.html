'우주인터넷망' 꿈꾸는 일론 머스크, '스타링크'에 300억달러 투입

일론 머스크 테슬라 최고경영자(CEO)의 말은 현실이 될 수 있을까. 그는 스페이스X의 위성 인터넷망 '스타링크'의 서비스 지역을 전 세계로 확대해 수십만명의 이용자를 확보하겠다는 목표를 제시했다. 스타링크 서비스 구축에만 300억달러(약 33조원)를 투자할 것으로 알려졌다.
